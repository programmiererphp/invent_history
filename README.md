## English

### Interactive Story Generator

An interactive web application that helps users create stories using AI assistance. The application supports voice input and text input, making storytelling accessible and engaging.

### Features

- **Interactive Storytelling**: Create stories with AI-assisted prompts and guidance
- **Multiple Input Methods**: Use voice recognition or text input to contribute to your story
- **Text-to-Speech**: Listen to your story being read aloud
- **Multiple Languages**: Interface available in English, Russian, and German
- **Customizable AI Models**: Choose from different AI models for story generation
- **Configurable Settings**: Customize prompts, themes, and other settings
- **Auto-start Feature**: Schedule automatic start of the application

### Requirements

- OpenRouter API key for AI functionality
- Modern web browser with JavaScript enabled
- Internet connection

### How to Use

1. Open the application in your web browser
2. Enter your API key in the settings
3. Click "Start Game" to begin creating a story
4. Follow the prompts and contribute to the story using voice or text input
5. Enjoy your creative storytelling experience!

### Technologies

- JavaScript for application logic
- Web Speech API for voice recognition and speech synthesis
- OpenRouter API for AI story generation
- Bootstrap for UI components

<a name="русский"></a>
## Русский

### Интерактивный Генератор Историй

Интерактивное веб-приложение, которое помогает пользователям создавать истории с помощью искусственного интеллекта. Приложение поддерживает как голосовой ввод, так и текстовый ввод, делая рассказывание историй доступным и увлекательным.

### Возможности

- **Интерактивное Повествование**: Создавайте истории с подсказками и рекомендациями от ИИ
- **Несколько Методов Ввода**: Используйте распознавание голоса или текстовый ввод для продолжения истории
- **Синтез Речи**: Прослушивайте созданную историю
- **Поддержка Нескольких Языков**: Интерфейс доступен на английском, русском и немецком языках
- **Настраиваемые Модели ИИ**: Выбирайте различные модели ИИ для генерации историй
- **Настраиваемые Параметры**: Настройка промптов, тем и других параметров
- **Функция Автозапуска**: Планирование автоматического запуска приложения

### Требования

- API ключ OpenRouter для функциональности ИИ
- Современный веб-браузер с поддержкой JavaScript
- Подключение к интернету

### Как Использовать

1. Откройте приложение в веб-браузере
2. Введите API ключ в настройках
3. Нажмите "Начать Игру" для создания истории
4. Следуйте подсказкам и вносите свой вклад в историю с помощью голоса или текста
5. Наслаждайтесь творческим процессом рассказывания истории!

### Технологии

- JavaScript для логики приложения
- Web Speech API для распознавания голоса и синтеза речи
- OpenRouter API для генерации историй с помощью ИИ
- Bootstrap для компонентов пользовательского интерфейса

## Deutsch

### Interaktiver Geschichtengenerator

Eine interaktive Web-Anwendung, die Benutzern hilft, Geschichten mit Hilfe künstlicher Intelligenz zu erstellen. Die Anwendung unterstützt sowohl Spracheingabe als auch Texteingabe, was das Geschichtenerzählen zugänglich und ansprechend macht.

### Funktionen

- **Interaktives Erzählen**: Erstellen Sie Geschichten mit KI-unterstützten Hinweisen und Anleitungen
- **Mehrere Eingabemethoden**: Verwenden Sie Spracherkennung oder Texteingabe, um zu Ihrer Geschichte beizutragen
- **Text-to-Speech**: Hören Sie sich Ihre Geschichte vorlesen an
- **Mehrere Sprachen**: Benutzeroberfläche verfügbar in Englisch, Russisch und Deutsch
- **Anpassbare KI-Modelle**: Wählen Sie aus verschiedenen KI-Modellen zur Geschichtengenerierung
- **Konfigurierbare Einstellungen**: Passen Sie Prompts, Themen und andere Einstellungen an
- **Auto-Start-Funktion**: Planen Sie den automatischen Start der Anwendung

### Anforderungen

- OpenRouter API-Schlüssel für KI-Funktionalität
- Moderner Webbrowser mit aktiviertem JavaScript
- Internetverbindung

### Bedienungsanleitung

1. Öffnen Sie die Anwendung in Ihrem Webbrowser
2. Geben Sie Ihren API-Schlüssel in den Einstellungen ein
3. Klicken Sie auf "Spiel Starten", um mit dem Erstellen einer Geschichte zu beginnen
4. Folgen Sie den Aufforderungen und tragen Sie per Sprach- oder Texteingabe zur Geschichte bei
5. Genießen Sie Ihr kreatives Storytelling-Erlebnis!

### Technologien

- JavaScript für Anwendungslogik
- Web Speech API für Spracherkennung und Sprachsynthese
- OpenRouter API für KI-Geschichtengenerierung
- Bootstrap für UI-Komponenten
